# LOTOFÁCIL

Liguagem utilizada: Python 3 (v3.8.2)  

### PACOTES UTILIZADOS  

- Pandas  
- Numpy  
- Keras  
- Scikit-learn

- Geração de pesos para cada dezena  
- Criação de jogos
